## §0. 前提と監査条項（本文書のすべての条項に優先する解釈原則）

### §0-1. 核と病の同居（明文化）。

本案の核＝相異制約の演繹（prop）は、公開開示からも流れ、無料推定を真実へ近づける。すなわち核の美点そのものが死因6の力学を強化しうる。この対立は論理的必然ではなく定量的バランスであるため、机上killではなく本工程で測る。D3（余剰ゼロ＝演繹最強）はこの対立が最も鋭く出るセットであり、それを承知で持ち越している。

### §0-2. Pass含意の限定。

本工程のPassが意味するのは「移動選択なし（固定順）・覗き非公開・賭け±1固定の条件下で、演繹コアが正の利ざやを持つ」ところまで。工程3（移動選択の導入）では、相手の車輪操作により「覗いた面が問われない」不発が構造的に復活する。工程3のkill基準には不発税の再検査を必ず含めること（本条項は工程3への予告として凍結する）。本工程の「不発ゼロ」は設計の性質ではなく固定順の人工物である。

### §0-3. 覗きの非公開。

覗きの実行事実・対象・結果は相手に一切公開しない。覗き公開が生む二次情報層（「あいつが覗いて強く賭けた」）は心理戦でありNPCシミュでは測れない（§6-14）。後の段の問いへ送る。

### §0-4. 両側最強実装（偽Pass・偽killの同時防止）。

無料側S-freeにも覗き側全戦略にも、真の生成モデルの厳密事後（§2-6の閉形式）を実装する。無料側が弱ければ偽Pass、覗き側が弱ければ偽killになる。実装格差の測定は本工程の失敗である。

### §0-5. 3項の事前見積もり義務。

死因6の完成形「買える情報の価値＝乖離×反転利得×命中率＞価格」の3項を§8に事前見積もりとして記載した。工程の結果がFAILの場合、死因の所在（核か骨格か価格か）を§8と突き合わせて特定する。

## §1. 単位語彙表（§15-10。判定指標の分母となる語は閾値と同格の凍結対象）

* 1ダイス ＝ D3の6値（§10で転記する定義）を面位置1..6へ一様ランダムに割り当てた順列1つ。F0-D条件では、各面の値をD3の6値プールから独立に復元抽出したもの（相異制約なし）
* 1ゲーム ＝ 1ダイスに対する4ラウンド。位置5・6は問われずに終了する（切詰め。理由は§2-2）
* 1ラウンド ＝ 覗きフェーズ → 賭けフェーズ → 開示・清算、の1巡
* 1測定セル ＝（条件×価格×対戦カード）1つに対するN=100,000ゲーム
* 勝敗 ＝ ゲーム終了時の総得点（賭け得点−覗き支出）の比較。同点は両者0.5勝として集計
* H/L ＝ D3の6値を値の大小で3値ずつに二分したクラスタ（大きい3値＝H。凍結）

## §2. 裸モデル仕様

### §2-1. 構成。

プレイヤー2人（対称）。共有の隠しダイス1個。両者は同じダイスの同じ面について賭ける。

### §2-2. ラウンド進行（固定順条件）。

ラウンドk（k=1..4）で位置kが問われる。順序は全員に既知（＝ロンデルの「到来時刻確定」の最小再現）。位置5・6は決して問われないことも既知。切詰めの理由: (a) k=5,6の自明化した賭け（枯渇平坦化）が市場シグナルを希釈するのを防ぐ (b) 「問われない位置」の存在が、締切知識に真の経済価値を与える（対照条件§3-2で覗きの不発リスクとして顕在化し、K-D3の検定に牙を持たせる）。

### §2-3. 覗きフェーズ。

各プレイヤーは同時・秘密裏に、価格cを支払って未公開かつ自分が未覗きの位置1つを覗ける（1ラウンドにつき最大1回。任意の未公開位置を選べる——問われない位置5・6も可）。結果は本人のみが知る（§0-3）。

### §2-4. 賭けフェーズ。

各プレイヤーは同時・秘密裏に、当該ラウンドで問われる面の値がHかLかを宣言する（棄権なし・強制二択）。的中+1点、外れ−1点。

### §2-5. 開示・清算。

問われた位置の値を公開し、賭けを清算。公開値は以後全員の共有知識（＝無料の演繹の原料）。

### §2-6. 事後計算の閉形式（全戦略共通・凍結）。

ダイスは既知6値の一様順列なので、任意の未知位置の値の周辺分布は「その観測者の既知値（公開開示＋自分の覗き）を除いた残余プール上の一様分布」に厳密一致する（工程1で検証済みの閉形式）。F0-D条件では各面独立のため、公開・覗きは当該面以外に一切波及しない（prop=0）。

### §2-7. 戦略の入力制限。

相手の覗き有無・賭け内容・累積得点は、いかなる戦略の入力にも含めない（§0-3の帰結。心理層は本工程の測定対象外）。

## §3. 条件と価格

### §3-1. 条件A: D3固定順（主条件）。

§2のとおり。

### §3-2. 条件B: D3ランダム順（K-D3対照＝締切なし）。

毎ラウンド、未公開位置（6箇所のうち残り）から一様ランダムに1つが抽選され問われる。抽選結果は開示の瞬間まで非公開。プレイヤーは「今ラウンド問われる位置がどれか」を知らずに覗き・賭けを行う（賭けは「今ラウンド公開される値がHかLか」に対して行う。事後は「残Hカウント／未公開数」で厳密に定義される）。4ラウンドで終了（2位置は未公開のまま）。覗いた位置が問われない可能性＝不発リスクがこの条件にのみ存在する。これは欠陥ではなく検定対象そのもの。

### §3-3. 条件C: F0-D固定順（K-D4対照＝演繹なし）。

ダイスのみF0-D（各面独立抽出）に差し替え、他は条件Aと同一。位置5・6への覗きはこの条件では純粋な無駄になる（propが存在しないため）。

### §3-4. 価格。

c ∈ {0, 0.25, 0.5, 1.0}。主判定はc=0.5のみ（K-D1・K-D4・K-D3）。c=0はK-D1'専用。c=0.25, 1.0は参考記録（§7）であり、いかなる判定にも使わない。

## §4. 戦略（全構成要素を凍結。§15-14）

共通仕様: 事後は§2-6の閉形式。賭け方向は P(H) > 0.5 ならH、P(H) < 0.5 ならL、P(H) = 0.5 ちょうどならH（凍結タイブレーク。恣意的だが対称条件で両者同一のため勝率に系統影響なし）。

* S-free（無料側基準）: 覗きなし。公開開示のみで厳密事後（＝無料演繹の最強形）。
* S-peek-imm: 毎ラウンド、当該ラウンドで問われる位置を覗く。ただしその位置の事後が自分にとって退化（1値に確定）していれば覗かない（支出節約）。条件B（ランダム順）では問われる位置が未知のため、「未公開・未覗き・非退化の位置」から一様ランダムに1つ覗く（内部乱数は§5-2の系統2）。候補が空なら覗かない。
* S-peek-marginal: 対象選択はS-peek-immと同一。ただし覗く前の当該賭けの事後P(H)が開区間(0.35, 0.65)にある場合のみ覗く（δ=0.15凍結。思考課金S3＝限界帯選別の系譜）。
* S-peek-prop: 問われないことが確定している位置（固定順では位置5・6）のうち、未覗き・非退化のものを番号の小さい順に覗く。両方消化済みなら覗かない。純粋にpropチャネルだけを買う戦略。条件Cでは定義上純粋な無駄（対照として意図どおり）。条件B（ランダム順）では「問われない位置」が存在しないため定義不能——条件Bでは走らせない（凍結。K-D3の選択集合から除外する理由）。

選択集合の凍結: K-D1・K-D1'・K-D4の「覗き系」選択集合＝{S-peek-imm, S-peek-marginal, S-peek-prop}。K-D3の選択集合＝{S-peek-imm, S-peek-marginal}。凍結後の戦略追加は§9の修正規律（開示つき1回限り）に従う。

## §5. 測定と統計

### §5-1. 対戦カード。

各測定セル＝「覗き系戦略X vs S-free」のN=100,000ゲーム。勝率＝Xの勝ち数（同点0.5）／N。

### §5-2. 乱数（共通乱数によるペアリング）。

seed_base=20260716。系統1（ダイス順列／F0-D抽出）: ゲームiのダイスは seed_base+i から生成し、全セルで共有（条件A・Bは同一順列、条件Cは同seedからのF0-D抽出）。系統2（条件Bの問われ位置抽選・S-peek-immの対象抽選）: 2×seed_base+i。系統3（ブートストラップ）: 3×seed_base。

### §5-3. 区間推定。

ゲーム単位のブートストラップ（B=10,000、パーセンタイル法、95%）。セル間の差（K-D3・K-D4）は共通ディールによる対応ブートストラップ（同iのゲームを対にしてリサンプル）。

### §5-4. 選択と判定の関係。

K-D4・K-D3で用いる戦略は、条件A・c=0.5の勝率点推定が最大のものを各選択集合から1つ選ぶ（凍結）。選択を条件A側で行うことは条件A側の超過を上振れさせ、K-D4/K-D3を発動しにくくする方向のバイアスを持つ——これは「ノイズで殺さない」側の保守性であり、承知の上で凍結する。K-D1の「最良」に伴う多重比較はPassを僅かに甘くするが、後段の工程が控えるため許容する（思考課金K1と同じ扱い）。

## §6. kill条件（OR——いずれか1つでも発動したら案Dをkillする）

判定はすべて凍結済みの数値・文言のみで機械的に行う。事後的な閾値再解釈・僅差の敗者復活を禁止する（§15-8, §15-9）。判定順は下の記載順（診断の深い順）とするが、全条件を計算・報告する（早期打切りしない）。

**K-D1'（情報の構造的無価値）**: 条件A・c=0で、選択集合3戦略すべてについて 対S-free勝率のCI95下限 ≤ 0.5 → 発動。含意: 無料でも情報が勝率に換金されない＝核の死が価格と無関係に確定。

**K-D1（市場不在・主判定）**: 条件A・c=0.5で、選択集合3戦略すべてについて 対S-free勝率のCI95下限 ≤ 0.5 → 発動。含意: 死因6 K1の再来。ただしK-D1'が非発動でK-D1のみ発動の場合、死因は「価格」に局在化する（§8と突き合わせ、死因6の「残された未検証枝」と同形式で記録する。記録のみ・行動には変換しない）。

**K-D4（演繹は市場に寄与しない）**: X*＝§5-4で選んだ戦略として、対応差CI95下限( exc_A(X*) − exc_C(X*) ) ≤ 0 → 発動。ここで exc_cond(X) ＝ 当該条件・c=0.5でのXの対S-free勝率 − 0.5。含意: 市場が立っていても、その利ざやは「隠し面ダイスなら何にでもあるimm」由来であり、相異制約の演繹＝本案の核は市場的に装飾。§0-1の「核と市場が逆向き」が実測された場合もここで発動する（exc_A < exc_C、すなわち演繹の公共財化が市場を純減させた場合）。

**K-D3（ロンデルは装飾）**: Y*＝§5-4で選んだ戦略（選択集合は{imm, marginal}）として、対応差CI95下限( exc_A(Y*) − exc_B(Y*) ) ≤ 0 → 発動。含意: 到来時刻の確定（＝ロンデルの最小機能）が覗きの経済価値を増やしていない。本案は「ロンデル×影」の空白を掘る案であるため、市場が立ってもコンセプトの死とする。測定範囲の限定（凍結）: 本条件が検定するのは締切の経済的側面（回収の計算可能性）のみであり、死因5が要求した心理的緊張は測れない。K-D3非発動を「締切圧の実証」と読むことを禁止する。

総合Pass＝4条件すべて不発動。含意は§0-2の限定に従う。Passした場合の次段は工程3（移動選択＝現在手の否定の導入＋不発税の再検査）のkill基準事前登録であり、design chatで議論する。

## §7. 参考記録（判定に使わない）

1. 転換率（K-D2相当・記録のみ）: 覗き結果が賭け方向を反転させたラウンドの割合（戦略別・条件別）
2. 覗き対象分布: 各戦略が覗いた位置の分布（特にS-peek-imm系が位置5・6を選ばないこと、条件Aでの純prop購入の実態）
3. kごとの層別: ラウンド別の正答率・覗き実行率・勝率寄与（工程1のkプロファイルと同形式）
4. 価格グリッド: c=0.25, 1.0での全セル勝率（市場の価格弾力性の観察）
5. S-freeの正答率のk推移: 無料演繹の強さの実測（§0-1の諸刃の定量化。条件Cとの差が「公共財化したprop」の量）
6. 同点率: 勝敗集計の健全性確認

## §8. 3項の事前見積もり（§0-5）

* 第1項（無料推定との乖離）: 工程1実測より、覗き1回の事後変化はimm 0.067〜0.125／prop 0.044〜0.125（D3・k=1..4帯）。二峰設計によりクラスタ確定が賭け方向の反転に直結しやすい
* 第2項（判断反転時の利得）: 賭け±1固定のため反転1回の期待振れ＝2点。ゲーム総得点レンジは賭け±4＋覗き支出（c=0.5で最大−2）であり、1反転がゲーム勝敗を覆す比重は大きい。既知の弱点: ±1固定は第2項を人工的に圧縮している。K-D1のみ発動しK-D1'不発の場合、第2項（賭け骨格）の再設計余地を死因記録に残す
* 第3項（命中率）: 条件Aでは位置1..4への覗きは命中率1（必ず問われる）、位置5・6は市場的不発だがprop価値を持つ。条件Bでは命中率が構造的に1未満——これはK-D3の検定対象そのもの（§3-2）

2. 実行時間の事前見積もりが4時間を超える場合、実装後・実行前に停止して報告する（nの削減は勝手に行わない）
