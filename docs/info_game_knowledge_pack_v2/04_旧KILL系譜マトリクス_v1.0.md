## 0. この文書の目的

本書は、過去案を思い出すための作品紹介ではない。

> **新案が、すでにKILLされた因果鎖の言い換えになっていないかを、設計開始前に判定するための停止表である。**

各案について、次を一意に固定する。

- どの親系統に属するか
- 前案の何を直そうとしたか
- 何が実際には改善したか
- どのKILLゲートで停止したか
- 一文での死因
- 他系統へ持ち出してよい部品
- 同じ親系統で再利用してはいけない部分

正式判定は六案すべて `KILL_AUDITED` とする。

```text
系譜B：観測者別私的価値を、本人の利益行動から読む

  │  修正：組単位シグナルと個体単位判断の解像度を一致させる
  ▼

  │  修正：可逆な逐次編集を捨て、不可逆な二択へする
  ▼

  │  結論：合理化すると全投入へ収束し、ヘッジ・ブラフ・人物読みが消滅。

  └─ 系譜B終了
```

## 正式な探索終了範囲

次の因果鎖を中心とする案は、新テーマ・新人数・新得点倍率であっても新系譜とは認めない。

```text
同じ対象を複数人が異なる私的価値で見る
↓
先行者が投資・配分・交換・選択・入札を行う
↓
後続者がその利益行動から先行者の私的価値を読む
↓
同じ対象群から採用・取得・交換する
```

# 5. 共通根本原因

六案を個別の失敗として扱うと、再び同じ系譜へ戻る。共通原因は次の五層である。

## 5-1. 有用な情報と、面白い情報の混同

- Oracle差が正である
- 行動変更率が正である
- 局所得点が増える

これらは必要条件でしかない。

『偏光コンベア』と『ステーク・リレー』は、情報が実際に有用でもゲームとしてKILLされた。

## 5-2. 方策・写像・閾値の復元を人物読みと呼んだ

相手を知ることが、次と同じなら人物読みではない。

```text
ルール表を覚える
固定閾値を推定する
既知方策のパラメータを当てる
```

『プロトコル市場』『分岐船団』『交差認証』で反復した。

## 5-3. 発信者か受信者のどちらか一方だけに選択があった

- 発信者が固定閾値で動く
- 受信者が情報を得ても個体順位を変えられない
- 発信者の選択が最終手で上書きされる

このいずれかで心理戦は消える。

## 5-4. 局所利益を最終競争利益と取り違えた

読みで採用判断が改善しても、

- 相手をより大きく利する
- 後続資源を失う
- 席順や最終手の効果に埋もれる
- 役割交代時に負ける

なら、読むことは勝利技能ではない。

## 5-5. ブラフの「可能性」と「合理的最善」を混同した

- 2–1配分がルール上可能
- 固定閾値から外れることが可能
- 嘘にコストがある

だけではブラフは成立しない。

> **正直、誤認、沈黙のそれぞれが、異なる局面で合理的最善にならなければならない。**

# 6. 系譜別の正式打切り判定

## 系譜A：固定方策・固定写像読み

対象：

1. プロトコル市場
2. 分岐船団

正式判定：`KILL_AUDITED`

再開禁止条件：

- 秘密をタイプ名、性格名、優先順位、写像、閾値へ置き換えるだけ
- 履歴を長くするだけ
- 方策数を3種から6種、12種へ増やすだけ
- NPCのプロフィール差を増やすだけ
- 推定UIを隠すだけ

新系譜と認めるには、相手が同じ状態でも複数の合理的行動を持ち、自分がどう読まれているかによって選択を変える必要がある。

## 系譜B：観測者別私的価値を利益行動から読む

対象：

1. パララックス・ロット
2. 偏光コンベア
3. 交差認証
4. ステーク・リレー

正式判定：`KILL_AUDITED / EXPLORATION_CLOSED`

現行の最大3案規則を遡及適用すれば、『ステーク・リレー』は4案目に当たり開始禁止だった。規則制定前の監査であるため歴史的記録として残すが、今後は同じ延長を許可しない。

再開禁止条件：

- 対象を遺物、モジュール、契約、試作品、カードへ変更する
- 2人を3人循環へする
- 投資キューブ数を3から4、5へ変える
- 私的価値倍率や得点配分を変える
- 逐次、同時、ラウンド制を入れ替えるだけ
- 「今回はブラフが起こるはず」と人物性を外部期待する

新系譜と認めるには、少なくとも因果鎖の中心を次のどれかへ交換する必要がある。

- 読む対象を私的価値ではなく、対立する長期計画・約束・危険許容などへ変える
- 発信行動と受信後の選択が、同じ対象の取得以外の関係を持つ
- 読みがなくても成立する葛藤を先に置き、情報をその価値反転にだけ使う
- 発信者と受信者が相互にモデルを更新し、同じ状態でも相手によって最善行動が変わる

# 7. 新案照合用・禁止判定

新案を提出する前に、以下へ回答する。

```text
1. 六案のうち最も近い案：
2. 同じまま残る因果矢印：
3. 交換した因果矢印：
4. なぜテーマ・人数・倍率変更ではないか：
5. 過去案の決定的KILLゲートを、どの構造で不成立にしたか：
6. ゲーム後に語られる具体的な一場面：
```

次のどれかに該当すれば、現役KILLゲートへ進む前に開始禁止とする。

- 「近い案なし」とだけ書き、比較を行わない
- 私的価値を利益行動から読む中心鎖が残る
- 固定方策を人物性と呼び換える
- 局所的な行動変更率だけを成功指標にする
- 最終手番、席順、役割交代を未検査のまま進める
- ブラフが可能であることだけを示し、最善局面を示さない
- 前案との違いがテーマ、数、倍率、UIだけ

```text
- 観測者別私的価値を投資・配分・交換・入札などの利益行動から読み、
  同じ対象群の採用・取得へ変換する系譜

テーマ、人数、倍率、公開範囲、資源名、カード名だけの変更は新系譜とみなさない。

開始する場合は、最も近い過去案、変更した因果矢印、残る共通部分、
過去案の決定的KILLゲートを無効化する最小反例を冒頭に記載すること。

監査中に過去案と同じ死因を発見した場合、救済ルールや次変形を提案せず停止し、
該当案、該当ゲート、同型である根拠を報告すること。
```

# 9. 最終総括

六案は、無意味な失敗の反復ではない。

```text
情報が行動を変えない
↓
静的には変えるが、動的対戦で勝てない
↓
シグナルと判断の解像度が合わない
↓
解像度を合わせると最終手番が支配する
↓
不可逆化すると固定閾値のセンサーになる
↓
配分自由度を与えると全投入へ収束し、読む側が最終競争で負ける
```

この進行により、現在は次を正式知見として持つ。

> **情報が有用であること、行動を変えること、局所利益を生むことは、心理戦ゲームの成立を意味しない。発信者と受信者の双方に生きた戦略選択があり、ブラフが合理的に最善となる局面が存在し、読みの利益が役割交代・席順・相手利益を含む最終勝率まで残り、最適プレイ時の勝敗を独自要素が支配して初めて、独自の情報ゲームとして成立候補になる。**
