# 2. Phase X：探索開始ゲート

## X0. 超系譜距離

次を記入する。

```text
中心動詞：
対象：
情報発生源：
情報後の行動：
利益経路：
最も近い停止単位：
一致する要素数：
```

### PASS

- 中心動詞が最も近い停止単位と異なる
- 5要素中3要素以上が異なる
- 閉鎖済み超系譜C・E・G・Hに該当しない

### KILL

- 「価値」を「意図」に変えただけ
- 追随を阻止へ変えただけ
- テーマ、人数、倍率、UIだけの変更
- 中心因果が`行動→隠れた内面→応答`のまま

## X1. 正の探索根拠

次の一つ以上が必要。

- 人間に機能すると確認済みの情報OFF土台
- 既存の生存ゲームから抽出した中心決断
- 情報が行動可能性を直接変えるデジタル固有操作
- 『魔女の調合』型の「成立済み葛藤＋情報変形」

「既知の死因に当たらなさそう」だけならHOLD。

# 3. Phase B：情報OFF土台ゲート

## B0. 情報なしでゲームか

秘密、履歴推定、人物差を全て外して記入する。

| 選択 | 得るもの | 失うもの | 正しくなる局面 |
|---|---|---|---|
| A |  |  |  |
| B |  |  |  |

### PASS

- 目的が一文で明確
- A/Bが同じ希少資源・時間・位置・勝ち筋を奪い合う
- 局面で優劣が反転
- 選択が次局面へ残る

机上表だけでは`PROVISIONAL`。B1なしでは次へ進めない。

## B1. 情報OFF人間マイクロテスト

- 2人以上
- 中心決断5～10回
- 各回で理由を一言記録

### PASS

- 2種類以上の選択が実際に候補となる
- 別手による具体的な結果差を説明できる
- 固定手へ即収束しない
- 何をしたいゲームか一致して説明できる

### KILL

- 一意手
- どちらでも同じ
- 目的不明
- 初見の理解不足だけが迷いを作る

# 4. Phase I：情報追加ゲート

## I0. 情報変換は一つか

記入する。

```text
情報OFFの中心決断：
追加する情報変換：
同じ決断のどの比較が変わるか：
情報なしでも残る葛藤：
```

### PASS

- 情報が元の決断を変形する
- 情報処理専用の別フェーズを作らない
- 人物内面の復元を中心にしない
- 情報は1種類・1作用に限定

### KILL

- 新しい帳簿づけ、暗記、写像復元を追加
- 情報を読めても元の決断が同じ
- 情報機構のために土台の目的を変更
- 事実を単独商品として売る

## I1. 情報ON人間マイクロテスト

B1と同じ参加者・土台で5～10回選ぶ。

### PASS

- 選択分布または理由が情報OFFから変わる
- 元の葛藤が残る
- 情報が別パズルを支配しない
- 成否を結果から説明できる

# 5. Phase S：場面証拠ゲート

## S0. OBSERVED_SCENE

次の形式で、実際のテストで起きた場面を書く。

> P1が【実際に見た状態】を見て、【実際に候補だったA】ではなく【B】を選んだ。  
> その結果【具体的な変化】が起き、本人は「【実際の発言】」と説明した。

### PASS

- ログ・盤面状態・本人発言がある
- 本人が結果を見る前に比較理由を説明できる
- 望んだ場面の創作ではない

### HOLD

- DESIRED_SCENEしかない
